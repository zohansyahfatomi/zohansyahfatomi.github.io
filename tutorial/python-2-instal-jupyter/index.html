Tutorial Python #2 : Instal Jupyter Notebook di Sistem Operasi [Linux]

Posted on 19 May 2020

Jupyter Notebook merupakan aplikasi berbasis web yang mengizinkan Anda untuk membuat web dan membagikannya kepada orang lain. Hal yang istimewa dari Jupyter Notebook adalah Anda dapat melakukan live-coding di platfrom dokumen tersebut. Tidak hanya itu, Anda dapat melakukan proses visualisasi data dan mencetak teks (berbasis markdown). Hal lain yang bisa dilakukan pada Jupyter Notebook antara lain :

- Data cleaning
- Data tranformation
- Simulasi Numerik
- Machine learning
- dlsb.

Dalam blog ini Anda akan mempelajari : - Bagaimana menginstal Jupyter Notebook pada Linux Ubuntu 20.04 LTS - Sedikit demonstrasi pada Jupyter Notebook.

Instalasi Jupyter Notebook pada Linux Ubuntu 20.04 LTS

  1. Memastikan Python 3.8.2 dan pip3 sudah terinstal Gunakan perintah
python3 --version

pada command-line untuk memverifikasi instalasi Python 3.8.2 dan

pip3 --version

untuk memverifikasi terinstalnya program pip pada sistem operasi Anda.

  1. Melakukan instalasi Jupyter Notebook Untuk melakukan instalasi Jupyter Notebook di Ubuntu 20.04 LTS Anda dapat memberikan perintah berikut ke command-line.
pip3 install --user jupyterlab
  1. Instalasi Jupyter Notebook sudah selesai.

Untuk mengakses Jupyter Notebook Anda dapat memanggilnya dengan memberikan kode comand-line sebagai berikut.

jupyter notebook

Kemudian pada browser Anda akan muncul jendela browser (Modzila Firefox) seperti gambar berikut.

Demonstrasi Jupyter Notebook pada Linux Ubuntu 20.04 LTS

  1. Untuk membuat projek baru, Anda menekan tombol New kemudian memilih opsi Python 3.

  2. Tampilan awal

  3. Mencoba Print Nama Anda Untuk mencoba apakah Jupyter Notebook yang Anda sudah terinstal dengan baik atau belum, Anda dapat mencoba membuat program sederhana dan mengeksekusinya. Program tersebut dapat mencetak nama Anda di cell keluaran.

print("Nama saya adalah Zohan Syah Fatomi")

Kesimpulan

Untuk mempermudah pembelajaran Python, Anda harus menguasai tools yang disediakan oleh Python termasuk Jupyter Notebook. Sebenarnya, masih banyak tools atau IDE (Integrated Development System) yang tersedia untuk Python seperti Pycharm, Anaconda, dlsb. Namun demikian, Jupyter Notebook memiliki kelebihan di antara yang lain diantaranya adalah Anda lebih mudah mendokumentasikan hasil pembelajaran Anda dan juga lebih bagus ketika Anda ingin memvisualisasikan data.
