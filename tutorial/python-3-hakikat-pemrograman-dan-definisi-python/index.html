Apa itu Program?

Di dalam ilmu komputer, program didefinisikan dengan suatu instruksi dari manusia yang ditujukan kepada komputer. Program dikonstruksi sedemikian rupa agar dapat membantu tugas dari manusia. Pada kenyataannya program tidak bisa bisa berdiri tunggal. Program selayaknya dikonstruksi dengan program-program yang lain sehingga membentuk perangkat lunak. Perangkat lunak inilah yang Anda sering gunakan di dalam komputer (PC) ataupun smartphone Anda.

Semisal Anda menggunakan perangkat lunak text editor, gedit. Program gedit digunakan biasanya digunakan untuk melakukan editing file berbasis teks atau juga bisa digunakan untuk menulis program.

Perlu diketahui juga bahwa suatu program memiliki konstruksi/model format pengeksekusian tertentu. Artinya setiap program dapat memiliki karakteristik pengeksekusian yang berbeda. Contoh program yang sederhana adalah program pencetak teks yang Anda buat di halaman sebelumnya yang bisa mencetak nama Anda.

Anda dapat membuat program yang lebih cerdas yaitu inputan nama diberikan pada saat program sedang berlangsung (running). Silahkan masukan kode Python sebagai berikut.

x = input("Masukan nama Anda : ")
print("Nama Anda adalah ",x)

Selain menjalankan program di Jupyter Notebook, Anda juga bisa menjalankan program Python di terminal Anda. Anda cukup memasukan kode perintah python3 kemudian terminal Python akan terpanggil (dalam hal ini disebut dengan RPEL). REPL kependekand dari Read Evaluate Process Looping.
