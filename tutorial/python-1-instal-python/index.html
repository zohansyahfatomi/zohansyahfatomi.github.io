Tutorial Python #1 : Instal Python di Sistem Operasi [Linux]

Di poskan pada: 18-05-2020

Python sebagai bahasa pemrograman tidak secara otomatis terinstal di dalam suatu Sistem Operasi (kecuali di Ubuntu yang secara default sudah terinstal Python versi 2.7.18rc1). Untuk menggunakan bahasa pemrograman Python maka Anda harus melakukan instalasi Python secara terpisah dari pada sistem operasi tersebut.

Dalam blog ini Anda akan mempelajari : - Bagaimana menginstal Python 3.8.2 pada sistem operasi Linux Ubntu 20.04 LTS - Melakukan verifikasi program Pythona apakah sudah terinstal pada sistem operasi Anda (dalam hal ini adalah Linux Ubuntu 20.04 LTS)

Proses Instalasi Python 3.8.2 pada Linux Ubuntu 20.04

  1. Mendownload file master di https://www.python.org/

  2. Ekstrak file Python*.tar.gz yang didapatkan dengan komando berikut.

tar -xvf Python-3.8.3.tar.xz

Atau seperti yang tampak pada gambar berikut.

  1. Jika Anda pertama kali melakukan instalasi Python lakukan perintah ini.
apt-get install build-essential
  1. Masukan perintah berikut pada command-line secara berurutan.
    ./configure

    make

    make test

    sudo make install
  1. Selesai. Python 3.8.2 dapat dieksekusi dengan menggunakan kata kunci python3.

Mengecek Hasil Instalasi Python 3.8.2 pada Linux Ubuntu 20.04

  1. Anda dapat melakukan pengecekan dengan menjalankan perintah berikut pada command-line.
python3 --version

Kesimpulan

Melakukan instalasi Python 3.8.2 pada sistem operasi Anda merupakan langkah awal dan esensial untuk belajar pada Python. Tanpa bisa melakukan instalasi Python 3.8.2 pada sistem operasi Anda, Anda tidak dapat memulai pembelajaran Python.
