Tutorial Python #4 : Interpreter vs Compiler

Di poskan pada: 24-05-2020

Compiler

Jadi, apa sih yang dimaksud dengan compiler? Well, sederhananya compiler merupakan suatu program yang digunakan untuk melakukan menerjemahkan (mentranformasikan) kode yang bersifat high-level (dekat dengan bahasa manusia) seperti Java, Javascript, Pyhthon dlsb. ke dalam kode low-level (dekat dengan bahasa mesin) seperti Assembly yang akan dieksekusi oleh komputer atau program lain seperti mesin virtual misalnya JVM (Java Virtual Machine).

Seperti yang disebutkan di atas bahwa contoh bahasa pemrograman yang menggunakan compiler adalah Java. Compiler dari pemrograman Java disebut dengan javac.

javac yang merupakan referensi dari java compiler digunakan untuk melakukan konversi kode Java ke dalam bentuk java-bytecode yang dapat dieksekusi oleh JVM. Adapun, proses konversi tersebut dapat diamati pada gambar tersebut.

Interpreter

Sebenarnya konsep bekerja pada interpreter mirip dengan proses pengerjaan pada compiler, yaitu melakukan transformasi dari kode sumber ke dalam kode mesin. Namun demikian, tidak seperti pada compiler yang menghasilkan suatu kode eksekusi (kode mesin), interpreter secara langsung mengeksekusi kode sumber (tanpa menghasilkan kode eksekusi).

Contoh bahasa pemrograman yang menggunakan interpreter adalah Python. Di Ubuntu 20.04 LTS, tnterpreter pada Python dapat Anda akses menggunakan kata kunci python (untuk Python 2.7) dan python3 (untuk Python 3.*).
