Python Dasar

Instalasi Python 3.11.4 di Ubuntu 22.04 LTS

Untuk melakukan instalasi Python di Ubuntu 22.04 LTS dapat dilakukan dengan :

  1. Mendownload python 3.11.4 dengan mengakses https://python.org/downloads

  2. Mengekstrak file Python-3.11.4.tar.xz dengan membuka terminal dan memasukan perintah:

zohan@ubuntu:~$ tar -xf Python-3.11.4.tar.xz
zohan
  1. Masuk ke dalam folder Python-3.11.4
zohan@ubuntu:~$ cd Python-3.11.4
zohan
  1. Melakukan cek konfigurasi
zohan@ubuntu:~$ ./configure
zohan

Proses pengecekan konfigurasi tersebut berguna untuk mengetahui apakah modul-modul yang dibutuhkan pada instalasi Python di Ubuntu sudah sesuai.

  1. Kompilasi
zohan@ubuntu:~$ sudo make
zohan

Proses ini akan mengkompilasi python di folder sekarang Anda berada.

  1. Kompilasi pada Ubuntu
zohan@ubuntu:~$ sudo make install
zohan

Proses ini akan mengkompilasi python di sistem Ubuntu.

  1. Kompilasi pada Ubuntu
zohan@ubuntu:~$ python3 --version
zohan@ubuntu:~$ pip3 --version
zohan@ubuntu:~$ which python3
zohan

Perlu diketahui bahwa pada Ubuntu terdapat jenis python yang sudah built-in terinstal meskipun Anda tidak menginstalnya yaitu python versi 2. Python yang sedang diinstal sekarang adalah Python versi 3.
